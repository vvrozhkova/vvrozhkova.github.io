Объединение множеств

(А∪B)(А \cup B)(А∪B) - множество, состоящее из всех тех элементов, которые принадлежат хотя бы одному из множеств А или В.
(А∪B)(А \cup B)(А∪B) = {x∣x∈Aилиx∈B}\left\{{ x|x \in A\:или\:x \in B} \right\}{x∣x∈Aилиx∈B}

Пример.
Даны два множества: A={1,2,4,6}A = \left\{ {1, 2, 4, 6} \right\}A={1,2,4,6} и B={0,3,4,6}B = \left\{ {0, 3, 4, 6} \right\}B={0,3,4,6}.
Найдем множество C=A∪BC = A \cup BC=A∪B.
C={0,1,2,3,4,6}C = \left\{ {0, 1, 2, 3, 4, 6} \right\}C={0,1,2,3,4,6}.

Пересечение множеств

(А∩В)( А \cap В )(А∩В) называется множество, состоящее из элементов, входящих как в множество А, так и в множество В.
А∩В={x∣x∈Aиx∈B}А \cap В =\left\{ {x | x \in A\:и\:x \in B} \right\}А∩В={x∣x∈Aиx∈B}.

Пример.
Даны множества A={1,2,4,6}A = \left\{ {1, 2, 4, 6} \right\}A={1,2,4,6} и B={0,3,4,6}B = \left\{ {0, 3, 4, 6} \right\}B={0,3,4,6}.
Найдем их пересечение: D=A∩B={4,6}D = A \cap B = \left\{ {4, 6} \right\}D=A∩B={4,6}.

Разность множеств

(A\B)( A\backslash B )(A\B) - множество всех элементов множества А, которые не содержатся в В.

A\B={x∣x∈Aиx∉B}A \backslash B = \left\{ {x \mid x \in A \:и\: x \notin B} \right\}A\B={x∣x∈Aиx∈/B};
B\A={x∣x∈Bиx∉A}B \backslash A = \left\{ {x \mid x \in B \:и\: x \notin A} \right\}B\A={x∣x∈Bиx∈/A}

Пример.
Даны два множества А и В. Найдем их разность.
A\B={1,2}A \backslash B = \left\{ {1, 2} \right\}A\B={1,2}; B\A={0,3}B\backslash A = \left\{ {0, 3} \right\}B\A={0,3}.

Симметричная разность

АΔВ=(А∪B)\(А∩B)={0,1,2,3,4,6}\{4,6}={0,1,2,3}А \Delta В = (А \cup B) \backslash (А \cap B) = \left\{ {0, 1, 2, 3, 4, 6} \right\}\backslash \left\{ {4, 6} \right\} = \left\{ {0, 1, 2, 3} \right\}АΔВ=(А∪B)\(А∩B)={0,1,2,3,4,6}\{4,6}={0,1,2,3}

Дополнение множества

Множество всех элементов, не принадлежащих А, но принадлежащих универсальному множеству.
А‾={x∣x∉Aиx∈Е}\overline А = \left\{ { x \mid x \notin A \:и\: x \in Е} \right\}А={x∣x∈/Aиx∈Е}.

Пример.
Пусть универсальное множество ЕЕЕ состоит из букв русского алфавита, ААА - множество гласных букв, тогда A‾\overline AA - множество согласных букв и букв ь и ъ.

Приоритет выполнения операций
