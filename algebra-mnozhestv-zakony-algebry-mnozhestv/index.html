Что такое алгебра множеств

Алгебра множеств в теории множеств — это непустая система подмножеств некоторого множества XXX, замкнутая относительно операций дополнения (разности) и объединения (суммы).

Законы алгебры множеств

- коммутативности

А∪В=В∪А;А∩В=В∩А;А \cup В = В \cup А; А \cap В = В \cap А;А∪В=В∪А;А∩В=В∩А;

- ассоциативности

А∪(B∪C)=(A∪B)∪C,А \cup ( B \cup C ) = ( A \cup B ) \cup C,А∪(B∪C)=(A∪B)∪C,A∩(B∩C)=(A∩B)∩C;A \cap ( B \cap C ) = ( A \cap B ) \cap C;A∩(B∩C)=(A∩B)∩C;

- дистрибутивности

  • пересечения относительно объединения:
    A∩(B∪C)=(A∩B)∪(A∩C);A \cap ( B \cup C ) = ( A \cap B ) \cup ( A \cap C );A∩(B∪C)=(A∩B)∪(A∩C);
  • объединение относительно пересечения:
    A∪(B∩C)=(A∪B)∩(A∪C);A \cup ( B \cap C ) = ( A \cup B ) \cap ( A \cup C);A∪(B∩C)=(A∪B)∩(A∪C);

- идемпотентности

A∪A=A,A∩A=A;A \cup A = A, A \cap A = A;A∪A=A,A∩A=A;

- действия с универсальным и пустым множествами

A∪∅=A,A∩∅=∅,A \cup \emptyset = A, A \cap \emptyset = \emptyset,A∪∅=A,A∩∅=∅,
A∪E=E,A∩E=A;A \cup E = E, A \cap E = A;A∪E=E,A∩E=A;
A∪A‾=E,A∩A‾=∅;A \cup \overline A = E, A \cap \overline A = \emptyset;A∪A=E,A∩A=∅;

- де Моргана

A∩B‾=A‾∪B‾,\overline {A \cap B} = \overline A \cup \overline B,A∩B=A∪B,
A∪B‾=A‾∩B‾;\overline {A \cup B} = \overline A \cap \overline B ;A∪B=A∩B;

- двойного дополнения

A‾‾=A.\overline{\overline A} = A.A=A.
